Tournament structure

Define divisions and schedule before play. Each division can use a different number of rounds and a different table count. Locked after you start the event.

Roster

Late arrival: you can add a player after start — they are marked “from R…” and first pair in the next round. If that round has no scores yet, open Pairings and use Rebuild pairings. Leave early / withdraw: their opponent is credited with the win for that table; later rounds omit them. The field toggles odd/even, so someone else may get or lose a bye next round.

    Tournament control

    Round 1 uses random pairings; later rounds use Swiss (similar records, no rematches when possible). Odd field sizes receive one bye per division per round.
